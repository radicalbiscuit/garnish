LICME Help:

Supported options:
    -l  list licenses available through licme
    -h  display this help file
    -a  specify an author name
    -y  specify a copyright year (default: current year)
    -f  force licme to overight the existing LICENSE

licme requires at least two options.  The first should be the name of the 
license you want to use. The second requirement is that you specify the
copyright holders using the -a option.


Usage examples:

    # Install the MIT license
    $ licme mit -a "Tim the Beaver"

    # Install the GPL3 license
    $ licme gpl3 -a "Richard Stallman" 

    # Install CRAPL license for crappy research code
    $ licme crapl -a "Matt Might"
